Requirements (Dev - Environment)
  • PHP 5.6 <
  • GitHub Account
  • MySQL 5.6 <
  • Apache2 Web-Server
  • Apache2-CakePHP-Libraries
  • Any PHP code editor
  • Any SQL editor (HeidiSQL, MySQL-WorkBench)
Installation
MySQL-Schema
  • Under the src there is a sql directory:
  • If you do not already have one create a root user in your MySQL with a password of 12345678.
  • Start off by creating your database run vetromedia.sql in your sql editor or from command line prompt.
  • First start by running the countries.sql
  • Then run the countriesdata.sql
  • After creating the countries in inserting the default data you can now create users
  • After the users.sql file you can now create the transactions table from transactions.sql
CakePHP Application
  • From the root of your web projects run
  • git clone https://github.com/fulelamakasi/vetromedia.git
  • This will pull the project to your local machine
  • Set up the local environment according to your machine standards i.e on Ubuntu the project would be in /var/www/vetromedia then setting up the vhosts etc.
  • Inside the cake project navigate your way inside config/app.php change the Email username and password if you're using gmail allow lesser apps on your gmail account.
  • Now you should be able to run the application locally.
